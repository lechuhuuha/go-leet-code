Brute force is checking every pair, which is O(n²) time and O(1) space
Optimized approach uses a hash map from value to index
For each number, I compute target - x and check whether that complement was seen before
If yes, I return the stored index and current index
Otherwise, I store the current value and its index
Complexity is O(n) time and O(n) space
